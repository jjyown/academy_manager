# 학부모 포탈 테스트 시나리오

## 📋 기능별 테스트 체크리스트

### 1. 페이지 로드 및 초기 상태

### 2. 학생 검색 기능

#### 2.1 이름으로 검색
```
1. 검색창에 학생 이름 입력 (예: "김철수")
2. 검색 버튼 클릭 또는 Enter 키
3. 결과 확인:
   - 학생 카드 표시
   - 학생명과 전화번호 표시
   - 출석률, 출석수, 지각수 통계 표시
```

- [ ] 학생 이름 검색 성공
- [ ] 부분 검색 작동 (예: "김" → "김철수" 검색됨)
- [ ] 검색 결과 없을 때 "검색 결과가 없습니다" 메시지
- [ ] 성공 알림 "○○○ 학생의 정보를 조회했습니다" 표시

#### 2.2 전화번호로 검색
```
1. 검색창에 전화번호 입력 (예: "01012345678")
2. 검색 버튼 클릭
3. 결과 확인
```

- [ ] 전화번호 검색 성공
- [ ] 부분 번호 검색 작동 (예: "1234" 포함 검색)

#### 2.3 오류 처리
- [ ] 빈 검색어 입력 시 "검색어를 입력해주세요" 메시지
- [ ] 존재하지 않는 학생 검색 시 "검색 결과가 없습니다" 메시지

### 3. QR 코드 스캔

#### 3.1 QR 스캔 모달 열기
```
1. QR 버튼 클릭
2. QR 스캔 모달 팝업
3. 카메라 권한 요청 (처음 사용 시)
```

- [ ] QR 버튼 클릭 시 모달 팝업
- [ ] 카메라 화면 표시
- [ ] 폐기 버튼(X) 작동
- [ ] ESC 키로 모달 닫기

#### 3.2 카메라 전환
```
1. "카메라 전환" 버튼 클릭
2. 카메라 방향 변경 (전방 ↔ 후방)
```

- [ ] 카메라 전환 버튼 작동
- [ ] 전방/후방 카메라 전환 성공

#### 3.3 QR 코드 인식
```
1. 학생 QR 코드를 카메라에 인식시킴
2. 자동으로 학생 정보 조회
3. 모달 자동 닫기
```

- [ ] QR 코드 자동 인식
- [ ] 모달 자동 닫기
- [ ] 학생 정보 표시
- [ ] 성공 알림 표시

### 4. 학생 정보 표시

#### 4.1 학생 카드
```
학생 카드 구성:
┌─────────────────────────┐
│ 학생명          [통계]   │
│ 📞 전화번호             │
├─────────────────────────┤
│ 출석률 95% | 출석 19회 │
│ 지각 1회               │
└─────────────────────────┘
```

- [ ] 학생명 표시
- [ ] 전화번호 표시
- [ ] 출석률 계산 정확
- [ ] 출석/지각/결석 통계 정확
- [ ] 카드 디자인 모바일 최적화

#### 4.2 출결 현황 섹션
```
최근 30일 출결 현황 리스트:
├ 날짜 (형식: MM-DD 요일)
├ 시간 (형식: HH:MM)
└ 상태 (✅ 출석 / ⏰ 지각 / ❌ 결석)
```

- [ ] 최근 30일 기록만 표시
- [ ] 날짜 역순으로 정렬 (최신순)
- [ ] 모든 상태 아이콘 표시
- [ ] 상태별 색상 구분 (초록/노랑/빨강)
- [ ] 출결 기록 없을 때 "최근 30일 출결 기록이 없습니다"

### 5. 종합 평가 섹션

#### 5.1 평가 입력
```
1. 평가 텍스트 입력 (최대 500자)
2. 글자수 카운터 실시간 표시
3. 저장 버튼 클릭
```

- [ ] 텍스트 입력 가능
- [ ] 500자 제한 작동
- [ ] 글자수 표시 정확 (0/500)
- [ ] 입력 시 글자수 실시간 업데이트

#### 5.2 평가 저장
```
1. 평가 내용 입력
2. "저장" 버튼 클릭
3. "평가가 저장되었습니다" 메시지 표시
```

- [ ] 저장 버튼 클릭 시 Supabase 저장
- [ ] 저장 성공 알림 표시
- [ ] 페이지 새로고침 후 평가 유지

#### 5.3 평가 초기화
```
1. "초기화" 버튼 클릭
2. 확인 팝업 나타남
3. "확인" 클릭 시 텍스트 지우기
```

- [ ] 초기화 버튼 작동
- [ ] 확인 팝업 표시
- [ ] 텍스트 삭제 실행
