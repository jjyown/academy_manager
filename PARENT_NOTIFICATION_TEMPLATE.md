## 🎯 발송 권장 사항

### 최적 발송 시간
- **평일**: 화~목요일 오전 9~10시
- **피크 시간 회피**: 월요일 오전, 금요일 오후
- **테스트**: 수요일 오전 9시

### 발송 빈도
- **첫 공지**: 즉시
- **리마인더**: 1주일 후
- **정기 안내**: 월 1회

### 대상 세분화
```
[ ] 신규 학생 부모 → 즉시 발송
[ ] 기존 학생 부모 → 주중 발송
[ ] 문의 많은 학생 → 우선 발송
[ ] 활동적인 부모 → 추가 기능 안내
```
